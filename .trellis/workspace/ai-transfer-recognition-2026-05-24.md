# AI记账转账识别功能优化

**日期**: 2026-05-24  
**目标**: 为AI记账功能添加转账识别能力，支持识别"微信零钱转20到中国银行"或"中国银行还花呗20"等转账场景

## 需求分析

### 用户场景
1. **资产间转移**: "微信零钱转20到中国银行"
2. **还款场景**: "中国银行还花呗20"
3. **其他转账**: "支付宝转账给微信100"

### 技术方案
利用现有的 `FinanceViewModel.transferAsset()` 方法，在AI解析时识别转账意图，返回特殊的JSON格式。

## 实现方案

### 1. 扩展 TransactionDraft 支持转账类型

在 `TransactionDraft.java` 中添加转账相关字段：
- `isTransfer`: 标识是否为转账
- `fromAsset`: 转出资产名称
- `toAsset`: 转入资产名称
- `discount`: 优惠金额（如还款优惠）

### 2. 更新 AI 提示词模板

在 `PromptTemplate.java` 中添加转账识别规则：
- 识别转账关键词（转、转账、还款、还、转入、转出等）
- 提取转出和转入资产
- 支持优惠金额识别

### 3. 扩展 TransactionDraftMapper

在 `TransactionDraftMapper.java` 中：
- 解析转账相关字段
- 匹配转出和转入资产账户

### 4. 修改 AiChatActivity 处理转账

在 `AiChatActivity.java` 中：
- 检测转账类型的 draft
- 调用 `FinanceViewModel.transferAsset()` 而不是普通记账
- 在卡片中显示转账信息

## 使用说明

### 如何使用转账功能

1. **打开AI记账**: 在应用中打开AI助手
2. **输入转账指令**: 说出或输入转账描述，例如：
   - "微信零钱转20到中国银行"
   - "中国银行还花呗20"
   - "支付宝转100到微信"
3. **查看卡片**: AI会生成转账卡片，显示：
   - 转账金额
   - 转出资产
   - 转入资产
   - 优惠金额（如有）
4. **确认入账**: 点击"确认入账"按钮完成转账

### AI识别示例

**输入**: "微信零钱转20到中国银行"

**AI返回JSON**:
```json
{
  "type": 2,
  "amount": 20.00,
  "category": "资产互转",
  "subCategory": "",
  "note": "微信零钱 -> 中国银行",
  "asset": "",
  "time": "2026-05-24 10:30",
  "isTransfer": true,
  "fromAsset": "微信零钱",
  "toAsset": "中国银行",
  "discount": 0
}
```

**卡片显示**:
- 标题: 转账 ¥20.00
- 转出: 微信零钱
- 转入: 中国银行
- 备注: 微信零钱 -> 中国银行

### 注意事项

1. **资产匹配**: AI会尝试匹配用户的资产账户名称，如果匹配失败，转账可能无法完成
2. **优惠处理**: 如果提到优惠（如"还款500，优惠10"），实际扣款金额 = 500 - 10 = 490
3. **还款场景**: 从资产账户还负债账户（如花呗、信用卡），fromAsset是资产，toAsset是负债
4. **不计入预算**: 转账记录不会计入预算统计（type=2）
5. **自动备份**: 转账完成后会自动触发WebDAV备份（如已启用）

## 技术细节

### 转账流程

1. **AI解析**: 识别转账意图，提取转出/转入资产和金额
2. **资产匹配**: 根据资产名称模糊匹配用户的资产账户ID
3. **生成卡片**: 显示转账信息供用户确认
4. **执行转账**: 调用 `FinanceViewModel.transferAsset()` 方法
5. **更新余额**: 
   - 转出账户余额减少（实际扣款金额 = 转账金额 - 优惠）
   - 转入账户余额增加（全额）
6. **生成记录**: 创建 type=2 的转账记录
7. **自动备份**: 触发WebDAV自动同步

### 与普通记账的区别

| 特性 | 普通记账 | 转账 |
|------|---------|------|
| type | 0(支出) / 1(收入) | 2(转账) |
| category | 用户分类 | "资产互转" |
| assetId | 单一资产 | 0（不关联） |
| 余额变化 | 单一账户 | 两个账户 |
| 预算统计 | 计入 | 不计入 |
| 卡片显示 | 分类+资产 | 转出+转入 |
