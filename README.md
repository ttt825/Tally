# Tally - 用户须知

## 📖 项目简介
Tally 是一款纯本地记账应用，所有数据仅存储在设备本地。支持支出、收入、借入、借出四种账单类型，提供日历月视图与年视图、明细筛选、统计分析等核心功能。
特色功能包括：照片记账与备份、多币种支持、二级分类管理、自动备份、微信/支付宝账单导入、桌面小组件、加班记录、自定义背景与主题、安全密码与生物识别锁等。

## 🛡️ 项目权限清单
- **相机**：用于拍照记账，为账单附加凭证照片
- **存储/文件**：用于数据导出导入、照片备份、自定义背景图片读取
- **通知**：提供实时状态反馈
- **悬浮窗**：用于快捷记账弹窗交互
- **生物识别**：用于应用锁的身份验证

## 💾 数据备份说明
- **导出数据（Json）**：导出全部数据，包括账单记录、分类信息、应用偏好设置等
- **导出账单（CSV）**：仅导出交易记录，适合用 Excel 等工具查看分析
- **导入数据（Json）**：从 Json 备份文件恢复全部数据
- **导入账单（CSV）**：仅导入交易记录
- **从微信/支付宝导入**：支持导入微信账单和支付宝账单，自动匹配分类
- **自动备份**：开启后，账单变动达到设定频次即自动导出 Json 全量数据到指定路径

## 🔒 隐私安全承诺
Tally 承诺完全不具备联网功能，未来也不会添加。所有数据仅存储在设备本地。
