# 用户须知

## 项目简介
Tally 是一个功能强大的记账应用，拥有自动记账、AI 智能记账、照片备份、WebDAV 云备份、自动续费提醒、多币种、关联资产等核心功能。所有数据存储在本地，可完全离线使用。WebDAV 云备份和 AI 记账功能需要联网，但完全由用户控制。

## 功能默认情况
- 记账助手设置："开启屏幕自动记账"关闭，"开启资产功能"开启
- 自动资产设置："开启自动资产"关闭
- 分类预设管理："开启二级分类"关闭
- 记账界面默认显示："结余"
- 货币单位设置："开启货币单位"关闭，默认"￥"
- 照片备注设置："开启照片备份"关闭。备份为纯本地存储（原理是复制一份至指定路径），推荐新建以"."开头（如 .Tally）的空白文件夹作为存储路径
- WebDAV 云备份："自动同步"关闭。需配置 WebDAV 服务器地址、用户名和密码，支持坚果云等网盘服务。开启自动同步后，每次记账、修改资产、修改分类等操作都会自动备份到云端
- AI 记账配置："启用 AI 记账"关闭。需配置兼容 OpenAI 的 API 服务（如 DeepSeek、通义千问等），支持文本、图片、语音识别。可设置 AI 分类关键字规则，自动应用分类
- 主题设置："跟随系统"
- 极简模式：关闭

💡 隐藏彩蛋：长按当前界面左上角的"用户须知"标题，可以取消激活高级设置。

## 项目权限清单
- 自启动：用于自动记账进程的后台保活。
- 无障碍：实现屏幕读取与自动记账的核心权限。
- 悬浮窗：用于 1 像素保活机制及记账时的即时弹窗交互。
- 通知：提供实时状态反馈，自动记账功能稳定后可手动关闭。

## 自动记账配置
使用"无障碍管理器"锁死 Tally 的无障碍权限。
"无障碍管理器"和"shizuku"在123链接里面的"辅助软件"，具体使用方法看B站视频。

## 无障碍原理
Tally的自动记账功能依赖于无障碍权限，下面的原理适用于"记账助手设置"、"自动记账设置"：

- 触发机制：只要在指定的应用内的屏幕出现了设定的关键字，就会触发自动记账（弹出悬浮窗、自动输入对应的金额）。例如，在"微信"这个应用设定关键字"微信"，那么每次打开"微信"都会触发自动记账。
- 注意事项：无障碍不识别图片。
- 资产关联："自动资产设置"是对自动记账功能的补充，如果没有设定相关的自动资产，那么记账会关联默认资产（长按资产设置默认资产）。

## WebDAV 云备份使用方法
WebDAV 是一种网络文件传输协议，可以将数据备份到支持 WebDAV 的网盘服务。

配置步骤：
1. 进入"设置" → "WebDAV 云备份"
2. 填写 WebDAV 服务器地址（如坚果云提供的地址）
3. 填写用户名和密码（坚果云需使用应用密码）
4. 点击"保存配置"
5. 点击"测试连接"验证配置是否正确
6. 开启"自动同步"开关（可选）

使用说明：
- 手动备份：点击"上传数据"按钮
- 手动恢复：点击"同步数据"按钮（会覆盖本地数据）
- 自动备份：开启后，每次记账、修改资产、修改分类等操作都会自动上传
- 备份内容：包含所有交易记录、资产账户、目标、分类设置等

推荐网盘：
- 坚果云（国内稳定，免费版每月1GB上传流量）
- Teracloud（日本服务，免费10GB空间）
- Box.com（国际服务，免费10GB空间）

## AI 记账使用方法
AI 记账功能可以通过文本、图片、语音识别账单信息，自动填写记账表单。

配置步骤：
1. 进入"设置" → "AI 记账配置"
2. 填写 API Base URL（如 https://api.deepseek.com）
3. 填写 API Key（从服务商获取）
4. 填写模型名称：
   - 文本模型（必填）：如 deepseek-chat
   - 视觉模型（可选）：如 deepseek-vision（支持图片识别）
   - 音频模型（可选）：如 whisper-1（支持语音识别）
5. 点击"测试连接"验证配置
6. 点击"保存配置"并开启"启用 AI 记账"

使用说明：
- 文本识别：在记账页面输入账单描述，AI 自动识别金额、分类等
- 图片识别：拍摄或选择账单照片，AI 自动提取信息
- 语音识别：语音输入账单信息，AI 转换为文字并识别
- 分类规则：在"AI 分类关键字规则"中设置关键字映射，提高分类准确性

推荐服务商：
- Longcat（国内，每天有免费额度，适合轻度使用）
- DeepSeek（国内，性价比高，1元/百万tokens）
- 通义千问（阿里云，免费额度较多）
- OpenAI（国际标准，需科学上网）

注意事项：
- API 调用需要消耗服务商的额度或费用
- 建议使用国内服务商，速度更快
- 首次使用建议先测试连接，确保配置正确
